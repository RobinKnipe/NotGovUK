# PageFooter

A very simple footer that can be used by internal applications but also extended
by the GovUK-style Footer component.

## When to use this component

This component is suitable for use in internal applications and for extending when creating custom footers.


## When not to use this component

Do not use the PageFooter component when you want the standard GovUK-style footer as there is a dedicated component for that.
